# User Management System

## PHP Setup Instructions

To set up the PHP environment for the User Management System, follow these steps:

1. Ensure you have PHP installed on your local machine. You can download it from [PHP's official website](https://www.php.net/downloads).

2. Clone the repository:
   ```bash
   git clone https://github.com/Syntax-Seeker/User_Management-System.git
   cd User_Management-System
   ```

3. then you can run this locally on you device using xmpp
